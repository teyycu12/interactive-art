# 數位轉譯角色空間互動系統 - 產品需求文件 (PRD)

## 1. 產品願景 (Vision)
[cite_start]透過視覺識別技術將「實體穿著」數位化，轉譯為具備個人風格的「插畫角色」，並在公共空間中透過群體演算法，將多人的社交行為轉化為一張集體共創的視覺敘事圖 [cite: 3]。

## 2. 應用場景 (Target Scenarios)
* [cite_start]**婚宴社交**：打破賓客間的尷尬，將「互不相識的賓客」轉化為「共創畫作的夥伴」 [cite: 4]。
* [cite_start]**網美餐廳**：讓顧客的參與成為空間裝潢的一部分，探討「使用者參與 (Co-creation)」的商業價值 [cite: 4]。
* **畢業典禮**：記錄全體畢業生的校園足跡，生成具備紀念價值的集體結晶。

## 3. 核心功能模組 (Core Modules)

### M1: 視覺特徵提取 (Perception)
* [cite_start]**特徵分析**：分析使用者服裝的主色調 (RGB/HEX)、輪廓 (Silhouette) 與姿勢 (Pose) [cite: 8]。
* **實時識別**：利用電腦視覺偵測使用者進入互動區域，並輸出結構化數據。

### M2: 插畫風格轉譯 (Translation)
* [cite_start]**組件化系統**：提供多種風格模板 (如手繪線條、日系Q版)，使用者可自定義配件 (如花束、學士帽) [cite: 12, 13]。
* [cite_start]**動態配色**：自動根據採集到的衣服顏色，為插畫角色進行即時換裝 [cite: 8]。

### M3: 群體動力交互 (Interaction)
* [cite_start]**自主社交行為**：角色進入投影牆後具備感知能力，當兩個角色靠近會觸發「打招呼」動作 [cite: 19, 20]。
* **群聚演算法 (Swarm Intelligence)**：遵循避障、對齊、凝聚三大原則，模擬自然的群體動態。

### M4: 集體記憶生成器 (Output)
* [cite_start]**自動構圖**：活動結束時，系統運用演算法將所有在場角色整理成一張美觀的「大合照插畫」 [cite: 24, 25]。
* [cite_start]**分享機制**：產出 QR Code 供使用者下載該次活動的視覺結晶 。
